# PROJECT_OVERVIEW.md

Version: 1.0

Status: Planning

Owner: Robert Bolgar

## Overview

QRLinkr is a standalone web application that allows anyone to generate high-quality QR codes instantly without advertisements, watermarks, account creation, or unnecessary complexity.

Its purpose is simple:

> Create beautiful QR codes for free.

QRLinkr is designed to be a complete product on its own while serving as the top-of-funnel into the TubeLinkr ecosystem.

## Vision

Build the fastest, cleanest, and most enjoyable QR code generator on the internet.

Every interaction should feel instant.
Every screen should feel intentional.
Every feature should reduce friction.

## Mission

Eliminate everything users dislike about existing QR code generators:

- No ads
- No watermark
- No account required
- High-quality downloads
- Modern interface
- Instant QR generation

## Core Principles

- Simplicity
- Speed
- Quality
- Trust
- Accessibility

## Relationship to TubeLinkr

QRLinkr solves the immediate problem:

"I need a QR code."

TubeLinkr solves the next problem:

"I want analytics, attribution, Smart Links, and Traffic Proof."

QRLinkr should never compete with TubeLinkr. It should naturally introduce users after they have successfully generated a QR code.

## Technical Philosophy

Version 1 should be:

- React
- Vite
- TypeScript
- Plain CSS
- Hosted on Vercel
- 100% client-side
- No backend
- No authentication
- No database
