# MVP_SCOPE.md

Version: 1.0

Status: Planning

Owner: Robert Bolgar

## Purpose

Define exactly what is included in QRLinkr Version 1 and, just as importantly, what is intentionally excluded.

The MVP should launch quickly while delivering an outstanding user experience.

## MVP Goal

A user should be able to:

1. Open QRLinkr
2. Choose a QR type
3. Enter their information
4. Optionally customize the QR code
5. Download a high-quality QR code
6. Leave without creating an account

## Included in Version 1

### QR Types

- Website URL
- Plain Text
- Email
- Phone
- SMS
- Wi-Fi

### Downloads

- PNG
- SVG
- Unlimited
- High Resolution
- Print Ready
- No Watermark

### Customization

- Foreground Color
- Background Color
- Transparent Background
- Margin
- Error Correction
- Output Size

### User Experience

- Dark theme
- Responsive design
- Fast performance
- Client-side generation
- Accessibility support

### TubeLinkr

Display a non-intrusive information card after successful QR generation introducing:

- Smart Links
- Analytics
- Attribution
- Traffic Proof

## Explicitly Excluded From Version 1

### Accounts

- Login
- Registration
- Profiles

### Cloud Features

- Saved QR Codes
- History
- Projects
- Sync

### Analytics

- Scan Tracking
- Click Tracking
- Reports
- Dashboards

### Business Features

- Teams
- Collaboration
- API
- Bulk Generation
- Payments
- Billing

### Marketing

- Advertisements
- Forced Signups
- Popups
- Exit Intent

These belong in future versions or TubeLinkr.

## Version 1.1 Candidates

- Copy QR Image
- Copy SVG
- More QR Types
- Additional Styling
- Logo Upload
- Rounded Modules

## Version 2 Candidates

- Dynamic QR Codes
- Saved Projects
- Teams
- API
- Bulk Generation

## TubeLinkr Exclusives

The following should remain exclusive to TubeLinkr:

- Smart Links
- QR Analytics
- Traffic Proof
- Placement Attribution
- Creator Hubs
- Video Attachments
- Branded Domains

## Definition of MVP Completion

Version 1 is complete when users can:

- Generate a QR code
- Customize it
- Download PNG
- Download SVG
- Complete the workflow without creating an account
- Discover TubeLinkr naturally afterward

## Product Rule

A successful MVP solves one problem exceptionally well.

Every feature should make QRLinkr a better QR code generator—not a more complicated application.
