# ROADMAP.md

Version: 1.0

Status: Planning

Owner: Robert Bolgar

## Purpose

Define the long-term evolution of QRLinkr.

The roadmap is organized into product phases rather than a simple feature list.

---

# Phase 1 — MVP

Goal:

Launch the best free QR code generator on the internet.

Includes:

- URL QR
- Custom Message QR (hosted page with URL-encoded content)
- Email QR
- Phone QR
- SMS QR
- Wi-Fi QR
- PNG Download
- SVG Download
- Color Customization
- Error Correction
- Responsive Design
- TubeLinkr Information Card

Success:

- Stable public launch
- Excellent UX
- Fast performance
- Positive user feedback

---

# Phase 2 — UX Improvements

Potential Features

- Better Mobile UX

- Additional Download Sizes
- Accessibility Improvements

# Phase 3 — More QR Types

Add support for:

- vCard
- Calendar
- WhatsApp
- FaceTime
- Google Maps
- PayPal
- Bitcoin
- App Store Links
- Social Profiles

Each QR type should receive its own SEO landing page.

---

# TubeLinkr Relationship

QRLinkr should never replace TubeLinkr.

QRLinkr creates QR codes.

TubeLinkr provides:

- Smart Links
- Analytics
- Attribution
- Traffic Proof
- Creator Workflows

The products should naturally complement one another.

# Guiding Principle

Every feature should answer one question:

"Does this make QRLinkr a better QR code generator?"

If not, reconsider whether it belongs in TubeLinkr instead.
